# Employee Management System

# Description

Spring Boot application to manage employees.

# Features

* Add Employee
* View All Employees
* View Employee by ID
* Update Employee
* Delete Employee
* Search by Department

# Steps to Run

1. Install MySQL and create database 'employee_Management'
2. Open project in IDE (IntelliJ / Eclipse)
3. Update DB username & password
4. Run Spring Boot application
5. Test APIs using Postman
